◆ throttle_enabled

int pool_stat::throttle_enabled

A var indicates whether the pool's throttle switcher is on or not

If it is none zero, it indicates that the throttle has been turned on, the pool will enject all requests delived by these APIs below:

stpool_task_queue stpool_add_routine stpool_group_add_routine.

user can set the throttle status by stpool_throttle_enable
