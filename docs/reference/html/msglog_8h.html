enum eFilterType

the working model of the global filter entry

Enumerator
eFT_discard 

Set the global filter entry to be a discarded entry

If level of the log message is not greater than the level that has been configured for the module, the message will be discarded.

eFT_allow 

Set the global filter entry to be a allowed entry

If level of the log message is not less than the level that has been configured for the module, the message will be allowed to be processed by the log system.

Write a debug message to the log system

MSG_log knows how to process the messages according to the global configurations

[in]fmtthe format of the user's arguments

EXPORT void MSG_log_set_handler ( msg_log_handler_t *  msgh)

Set the global log message handler (Gloal::@msgh)

the message handler will take the ownership of the right to handle the debug messages. it means that all of the other global configurations will be ignored if the global log message handler is not NULL.

Parameters
[in]msgh
Returns
None
Note
If you want to process the debug messages in your own way, MSG_log_set_handler must be called before your first calling MSG_log or MSG_log2
